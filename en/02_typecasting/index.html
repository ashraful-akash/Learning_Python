    Lesson 02: Type Casting

    üìò Topic Overview

    In this lesson you will learn:

    • What type casting means in Python
    • How to check the data type of a variable using type()
    • How to convert between basic data types:
    • string ‚Üî integer
    • integer ‚Üî float
    • number ‚Üî string
    • string ‚Üî boolean
    • Common mistakes and important rules when converting types

    The examples below are taken from the file 02_typecasting.py.

    üîç Checking variable types with type()

    The type() function is used to find the data type of a variable.

    name = "Ashraful"
age = 25
gpa = 3.25
is_student = True
print(type(name))
print(type(age))
print(type(gpa))
print(type(is_student))

    Output:

    <class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>

    Explanation:

    • "Ashraful" ‚Üí str
    • 25 ‚Üí int
    • 3.25 ‚Üí float
    • True ‚Üí bool

    üîÑ What is Type Casting?

    Type casting means converting one data type into another.

    Python provides built-in functions for this:

    • int() ‚Üí convert to integer
    • float() ‚Üí convert to float
    • str() ‚Üí convert to string
    • bool() ‚Üí convert to boolean

    üî¢ Converting float to int

    gpa = int(gpa)
print(gpa)

    Output:

    3

    Important:

    • Converting a float to an integer removes the decimal part (it does not round).
    • 3.25 becomes 3.

    üî∏ Converting int to float

    age = float(age)
print(age)

    Output:

    25.0

    Explanation:

    • The integer 25 becomes the float 25.0.

    üî§ Converting number to string

    age = str(age)
age = age + "1"
print(age)
print(type(age))

    Output:

    25.01
<class 'str'>

    Explanation:

    • After converting age to a string, "25.0" + "1" results in string concatenation.
    • Python does not perform numeric addition on strings.

    ‚úÖ Converting string to boolean

    name = bool(name)
print(name)

    Output:

    True

    Rules for bool():

    • Any non-empty string ‚Üí True
    • Empty string "" ‚Üí False

    Example:

    bool("")      # False
bool("Hi")    # True

    üõ† Common mistakes & tips

    • Converting invalid strings to numbers causes errors:

    python int("abc") # ‚ùå ValueError * String addition is not numeric addition:

    python "2" + "3" # "23" * Order matters when casting:

    python int("25") # ‚úÖ int(25.9) # ‚úÖ int("25.9") # ‚ùå ValueError * Always check types using type() when debugging.
